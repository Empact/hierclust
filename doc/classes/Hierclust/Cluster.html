Class Hierclust::Cluster
In: lib/hierclust/cluster.rb
Parent: Point

A Cluster represents a collection of Points. A Cluster has it‘s own coordinates that are the mean of the coordinates of it‘s points. Because a Cluster has coordinates, it can act as a Point and therefore be included in other Clusters.
